# Nocterra

Nocterra is a general-purpose, freedom-respecting content management system for websites.

It is built for fast, accessible, privacy-friendly and maintainable public, professional, non-profit and commercial websites without unnecessary database or admin backend overhead.

## What Nocterra is for

Nocterra is a general-purpose CMS. It can be used for business websites, restaurants, professional practices, foundation websites, project websites, editorial websites, documentation and custom web applications.

Nocterra is chosen when a website should remain understandable, portable and under the maintainer's control, without depending on a large visual editor, shortcode layer, plugin stack, theme subscription cycle or public administration backend.

Privacy and security are part of the normal design of Nocterra. They do not mean that Nocterra is only intended for privacy-critical organisations.

## History

Nocterra has a longer project history than the current name alone shows.

The system started as BackPage, a server-side publishing system created as a practical answer to desktop web-authoring tools such as Microsoft FrontPage. It was later renamed Synergos, when the project explored a more interactive publishing model with database-backed editing and early website ordering functionality.

After the 2019 server failure, remembered across the foundation as the Great server crash™, Synergos was not restored in its existing form. The CMS was renamed Nocterra and reprogrammed from the ground up.
